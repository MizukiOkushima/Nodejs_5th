# Nodejs_5th

### バージョンについて
Node.js v22.1.0<br>

### 環境構築
package.jsonファイルの生成<br>
`npm init`<br>
<br>
expressのインストール<br>
`npm i express`<br>
<br>
axiosのインストール<br>
HTTPのリクエストを送るときに使えるモジュール<br>
`npm i axios`<br>
<br>
cheerioのインストール<br>
スクレイピングをするモジュール<br>
`npm i cheerio`<br>
<br>
nodemonのインストール<br>
`npm i nodemon`<br>
<br>
package.jsonのscriptsの編集<br>
`"scripts": {
    "dev": "nodemon server.js"
}`<br>
